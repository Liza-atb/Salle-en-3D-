# Salle de TP en 3D

Projet réalisé dans le cadre du module d’infographie, en **Processing**.

Ce projet consiste à modéliser notre salle de travaux pratiques en 3D.  
La scène comprend :

- La modélisation complète de la salle
- Le mobilier (tables, chaises, etc.)
- Une caméra interactive
- La possibilité de se déplacer dans la scène

L’objectif était de mettre en pratique les notions d’infographie 3D avec Processing :
- Modélisation d’objets
- Gestion de la caméra
- Transformations (translations, rotations, échelles)
- Gestion de la scène 3D
